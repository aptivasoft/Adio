Reports

There are several standard reports listed in the system.  The reports are self-explanatory. Search options were provided in each report. In each report search can be done based on Clinic Organization, based on Facility and between specific dates like user can provide start date and end date.  The reports that are currently part of the application are:

-       Prescription History: Provides total number of prescriptions in the current month. Report provides classification appropriately in such a way that it displays the Clinic Organization name first and list all the related facilities of the clinic. Columns include details like the Patient Name, doctor name who has entered the prescription, Prescription date, suggested drug name, quantity and refills. And after listing the details of the prescriptions, total number of prescriptions of the facility is calculated and at the last total number of prescriptions for the clinic is calculated, finally after all clinics, grand totals of all the clinics is also calculated.
