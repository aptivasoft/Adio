Pharmacist Home Page:

Authorized Pharmacist can have access to this Home Page.  The Pharmacist Home Page displays the following information: 

·         Today’s Med-Requests

·         Pending Med-Line Issues

·         RxQueueList

·         Upcoming Events

·         Announcements/Messages

Today’s Med Requests:

It shows all the Pending and Action Taken med requests for the pharmacists.  

Description: Medication Name, Dosage, Quantity and Dosage.

Doctor: Patient’s Doctor Name.

Action Taken by Doctor: Shows the action taken for the med request.

 

Pending Med-Line Issues:

It shows the pending med-line issues identified for the pharmacist.

Date: Date indicates the Med-Line Issue date.

Patient: Patient Name.

Description: Med-Line Issue Description.

Action: When Pharmacist clicks on Action Link the following Med Issue popup window will be displayed:

The Pharmacist can enter a comment in response to the Med Issue.

 

RxQueueList:

The Rx Queue List displays list of Rx for the Patient’s order by Clinic-Facility Location.

By default it shows today’s Rx from All Organizations and All Locations for the patients with Transmitted Status. 

The following picture shows an Empty List of Rx Queue when there is no Rx in that date.

The following picture shows list of Rx from All organization and All Locations with status Transmitted.

Patient Name: The Patient Name column displays Clinic/Organization Name and the Location under the clinic and Doctor Name - Patient’s name.

Rx Type: It displays 3 types of Rx.

    • Regular
    • Sample
    • PAP

Quantity: Indicates the quantity of drug prescribed.

User: User Name that indicates who entered that Rx.

Time: Indicates time of Rx entry.

Order Status: It displays one of the following Rx Status in the list.

      • Transmitted
      • Received
      • Dispatched
      • Discontinued

 When Pharmacist clicks on Print Icon the Order Status will be updated as ‘Received’.

Upcoming Events:

It shows the List of Event Description, Date and Time of Event.

 

Announcements/Messages:

 

It shows the List of Announcement/Messages posted by Users.

 

 

It displays the following information:

 

Sr.No: Indicates the Serial Number of the Announcement/Message.

 

Date:  Indicates the Date of Announcement/Message.

 

Message: Announcement/Message Description.

 

ANNC. By: Announcer’s Name.
