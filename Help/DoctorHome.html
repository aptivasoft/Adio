Doctor Home Page:

Authorized Doctors can have access to this Home Page.  The Doctor Home Page displays the following information based on the Doctor’s Facility Location in a particular clinic: 

·         Pending Med Requests

·         Pending Med-Line Issues

·         Today’s Activity for CLINIC

Pending Med Requests:

It shows all the pending med requests order by Date in descending order.  

Date: Date indicates the Med Request’s date.

Patient: Patient Name registered in the Doctor’s Clinic – Facility Location.

Medication: Medication Name, Dosage, Quantity and Dosage.

Doctor: Doctor Name registered in the Clinic – Facility Location.

Action: Action has two types: Approve and Deny.

Approve: When Doctor Clicks on Approve link the following RxApproval popup window will be displayed.

Doctor can enter/change the quantity and SIG. Can make a note to the Pharmacist and click on Save button to Approve the Med Request.

 

 

Deny:  When Doctor clicks on Deny link the following popup window will be displayed.

Doctor cannot enter/change the quantity and SIG. Can make a note to the Pharmacist and click on Save button to deny the Med Request.

Pending Med-Line Issues:

Date: Date indicates the Med-Line Issue date.

Patient: Patient Name registered in the Doctor’s Clinic – Facility Location.

Description: Med-Line Issue Description.

Doctor: Doctor Name registered in the Clinic – Facility Location.

Action:  On clicking Action link the following Med Issue popup window will be displayed.

Doctor can enter a Comment and click on save button to respond for pending the Med Issue.

 

 

 

Today’s Activity for CLINIC:

It shows the following activity for the CLINIC.

No. of Prescriptions: Total number of New Prescriptions.

No. of Refills: Total number of Refills.

No. of Med Requests: Total number of Med Requests made.

No. of Med-Line Issues: Total number of Med-Line Issues.
