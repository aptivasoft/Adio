Employee Information

Employee Profile web page allows the user to maintain employee data. In this form user will be able to provide/insert employee information.  Search option is provided to search for an employee providing employee name in the format “Last Name, First Name”. This search textbox have the auto-fill capability. Search also allows user to edit the information of employee.  System automatically retrieves employee names when user attempts to enter search data in this textbox

Image M4-1

1.       Provide Employee First Name and Last Name

2.       Select Facility Name from the auto fill capability of the textbox provided for location

3.       Select Hire-date from the employee

4.       Provide supervisor for the employee from the auto-fill textbox. Supervisor1 and Supervisor2 cannot be same

5.       Provide Title for the employee. This should be selected from the auto-fill textbox

6.       To add new title to the list of titles check this checkbox.

7.       Select the status of the employee

8.       Select the employee role from the dropdown list

9.       Click on save to insert the employee details

10.   Click on cancel to clear the data from the textboxes and selection from the dropdown list

11.   Click on this to view the list of all employees

Provide title and click on save to create new title
