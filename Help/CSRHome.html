CSR Home Page:

Authorized CSRs can have access to this Home Page.  The CSR Home Page displays the following information based on the CSR’s Facility Location in a particular clinic: 

·         Scheduled Appointments for Today

·         Actions Taken by Doctors

·         Announcement/Messages

·         Upcoming Events

·         Today’s Activity for CLINIC

Scheduled Appointments for Today:

It shows the Patient‘s appointment details with the doctor for today. It shows Appointment Time, Patient Name, and Doctor Name.  Clicking on Patient Name will redirect to Patient Profile Page showing the details of the selected Patient.

Actions Taken by Doctors:

It shows Med Requests and Med Issues which are Pending as well as Action Taken by doctors.  It displays the following information for both pending and action taken on med requests and med line issues or call log issues:

Med Request in case of Med Request else shows Med Issue.

Description: Contains medication name, dosage, quantity and sig.

Doctor: Doctor Name

Action Taken:  Pending, Approved, Denied for corresponding action taken on med requests or med line issues.

 

 

 

 

Announcement/Messages:

It shows the Announcements or Messages posted by the users.

Sr.No:  Serial Number of announcement.

Date: Announcement Date.

Message: Announcement Description.

ANNC. By:  Announcer’s Name.

Up Coming Events:

It shows the upcoming events by date and time and the event name.

Today’s Activity for CLINIC:

It shows the following activity for the CLINIC.

No. of New Patients: Total number of New Patients registered.

No. of Prescriptions: Total number of New Prescriptions.

No. of Sample Rx: Total number of Sample Rx.

No. of Med Requests: Total number of Med Requests made.

No. of Patient Calls: Total number of Patient Calls.
